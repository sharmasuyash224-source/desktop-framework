# Changelog

All notable changes to Desktop Framework are documented in this file.

---

## v1.0.0

### Added

* Unified `desktop` CLI
* Modular architecture
* Plugin-based theme engine
* Pipeline engine
* Wallpaper framework
* Session manager
* Waybar module
* Hyprlock module
* Rofi module
* SwayNC module
* Powermenu module
* Wallpaper automation
* Dynamic Wallust integration
* Backup command
* Update command
* Validation tools
* Desktop doctor
* Desktop status

### Changed

* Converted traditional dotfiles into a modular desktop framework.
* Standardized module lifecycle.
* Unified desktop management through a single CLI.

### Release

Initial stable public release.
