# Installation

## Requirements

* Arch Linux or an Arch-based distribution
* Hyprland
* Wayland
* Git
* Bash
* systemd
* Wallust
* Hyprpaper
* Hyprlock
* Waybar
* Rofi
* SwayNC

## Clone

```bash
git clone https://github.com/<username>/desktop-framework.git

cd desktop-framework
```

## Install

```bash
chmod +x install.sh

./install.sh
```

## Verify

```bash
desktop doctor
desktop status
```

## Update

```bash
desktop update
```

## Backup

```bash
desktop backup
```

## Uninstall

```bash
./uninstall.sh
```
