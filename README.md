# Desktop Framework

> A modular desktop framework for Hyprland with a unified CLI, pipeline-driven workflows, and plugin-based theming.

Desktop Framework transforms a traditional Hyprland configuration into a structured, modular, and maintainable system.

Instead of managing independent configuration files and scripts, Desktop Framework provides a unified command-line interface, modular components, a plugin-based theme engine, and pipeline-driven automation for managing the desktop.

---

## Features

* Unified `desktop` CLI
* Modular desktop architecture
* Pipeline-driven wallpaper engine
* Plugin-based theme engine
* Dynamic Wallust integration
* Waybar module
* Hyprlock module
* SwayNC module
* Rofi launcher module
* Powermenu module
* Session management
* Built-in diagnostics (`desktop doctor`)
* Desktop backup and update commands
